# 🏗️ معماری فنی

## معماری سه‌لایه:
Presentation (Dashboard, API, WebSocket)
↓
Business Logic (Agents, Strategies, Risk)
↓
Data & Integration (Database, Cache, MT5)

## تکنولوژی‌ها:
Backend: Python 3.11+ | API: FastAPI | DB: PostgreSQL | Cache: Redis | Queue: RabbitMQ

## پایگاه داده (11 جدول):
users | accounts | trades | signals | performance | logs | licenses | api_keys | candles | ticks | sentiment

## امنیت (5 لایه):
Network: SSL/TLS | Auth: JWT, 2FA | Encryption: At-Rest, In-Transit | Audit: Logging

## مقیاس‌پذیری:
Level 1: Single Server | Level 2: Load Balancer + Replicas | Level 3: Kubernetes + Sharding
